# 🚀 Realtime Code Editor

A modern, real-time collaborative code editor built with **React.js**, **Node.js**, **Socket.IO**, and **Monaco Editor**. Multiple users can join the same room, write code together, view live updates instantly, and collaborate seamlessly.

---

## 📌 Features

✨ Real-time collaborative code editing

👥 Multiple users can join the same room

⚡ Instant code synchronization using Socket.IO

🟢 Live typing indicator

💻 Monaco Editor (VS Code Editor)

🌐 Room-based collaboration

📋 Copy Room ID with one click

👨‍💻 Active users list

🎨 Beautiful and responsive UI

🌙 VS Code Dark Theme

🔄 Language synchronization across all users

🚪 Join and Leave Room functionality

⚙️ Fast communication using WebSockets

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/realtime-code-editor.git

cd realtime-code-editor
```

---

### Install Backend Dependencies

```bash
npm install
```

---

### Install Frontend Dependencies

```bash
cd frontend

npm install
```

---

## ▶ Run the Project

### Start Backend

```bash
npm run dev
```

or

```bash
npm start
```

---

### Start Frontend

```bash
cd frontend

npm run dev
```

---

Open

```
http://localhost:5173
```

## 📸 Application Workflow

1. Enter your Username.
2. Enter Room ID.
3. Join the room.
4. Invite others using the Room ID.
5. Collaborate in real time.
6. View active users.
7. See who is typing.
8. Change programming language together.
9. Leave the room when finished.
